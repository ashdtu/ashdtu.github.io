Research

User Interaction Lab,Aalto University [June'17-Sep'17]

Gaussian Process RL framework for Visual Menu Search Task

Paper
  • Main goal was to understand the search behavior while the user interacts with a visual UI menu
  • The problem could be split into two parts: designing a cognitive model of the user and inference of model parameters from actually observed eye tracking data
  • Our PO-MDP model of Menu environment uses Gaussian Process posterior to update Value function
  • Investigated several exploration strategies for the sparse rewards learner.

Robotics Research Center, IIIT-Hyderabad (Dec 2016 - Jan 2017)

Max margin IRL agent generates safe navigation trajctories for ORB-SLAM

Paper
  • We have modeled our reward function as a linear combination of 5 features and thus solving a Max-margin type optimisation problem learning the weights of these features.
  • Decision Tree Regressor used for approximation for continous state space.
  • Our expert demonstrations are itself given by a Q-learner type RL agent with a handcrafted reward function(based on these 5 features).
  • The optimisation problem can be solved by quadratic programming formulation and we are using “cvxopt” python library for that.
